# assets/

Image and media assets live here. The app references them locally so nothing depends on an expiring URL.

Drop these in:

- **hero.png** : the clean café-terrace hero illustration (no UI, no title text; the garbled "Café de Nuit Étolée" sign is the kept easter egg). This is the painting you generated. Square 1024x1024; the build positions it to keep the cat in frame on wide screens.
- **favicon.png** (or .ico)
- **og-image.png** : social / share preview image (a crop of the hero works beautifully)
- **chanson.mp3** (optional, STRETCH) : a royalty-free / CC0 French ambient track

Only `hero.png` is needed for v1. The rest are nice-to-haves.
